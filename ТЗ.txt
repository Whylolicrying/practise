Приложение "Вычисление выборки" предназначено для анализа данных, позволяя пользователям проводить выборку из заданного набора данных и вычислять статистические показатели.

▎2. Цели и задачи

• Обеспечить пользователям возможность загружать данные в формате CSV.

• Позволить пользователям выбирать параметры выборки (например, размер выборки, условия фильтрации).

• Вычислять и отображать основные статистические показатели (среднее, медиана, стандартное отклонение и т.д.) для выбранной выборки.

▎3. Функциональные требования

1. Загрузка данных:

   • Пользователь может загрузить файл в формате CSV.

   • Приложение должно обрабатывать ошибки загрузки (например, неверный формат файла).

2. Выбор параметров выборки:

   • Пользователь может задать размер выборки.

   • Возможность фильтрации данных по заданным критериям (например, по диапазону значений).

3. Вычисление статистических показателей:

   • Приложение должно вычислять следующие показатели:

     • Среднее значение

     • Медиана

     • Стандартное отклонение

     • Минимальное и максимальное значения

4. Отображение результатов:

   • Результаты вычислений должны отображаться на экране в удобном для пользователя формате.

   • Возможность экспорта результатов в формате CSV или PDF.

▎4. Нефункциональные требования

• Приложение должно быть кросс-платформенным (Windows, macOS, Linux).

• Интерфейс пользователя должен быть интуитивно понятным и доступным.

• Время отклика приложения при загрузке и обработке данных не должно превышать 5 секунд для наборов данных до 10,000 строк.
